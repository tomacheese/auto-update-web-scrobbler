# AGENTS.md

## 目的
このファイルは、AI エージェント共通の作業方針を定義します。

## 基本方針
- 会話言語: 日本語
- コード内コメント: 日本語
- エラーメッセージ: 英語
- コミット規約: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) に従い、説明文は日本語とする

## 判断記録のルール
判断を行った際は、以下の内容を明示するように努めてください：
- 判断内容の要約
- 検討した代替案
- 採用理由
- 前提条件・仮定・不確実性

## 開発手順（概要）
1. プロジェクトの構造と目的を理解する
2. `pnpm install` で依存関係をインストールする
3. 既存の規約に従って変更を実装する
4. テストと Lint/Format を実行し、問題がないことを確認する
5. 変更内容を具体的に説明し、PR を作成する

## セキュリティ / 機密情報
- API キー、パスワード、トークンなどの認証情報を Git にコミットしない
- ログに機密情報（個人情報、トークン等）を出力しない
- 環境変数や設定ファイルを適切に扱う

## リポジトリ固有
- Web Scrobber の LevelDB を直接操作するため、データの破損に細心の注意を払う
- ネットワーク通信を行う際は、エラーハンドリングを適切に行う
- 設定ファイル `data/config.json` はユーザー環境に依存するため、バリデーションを徹底する
